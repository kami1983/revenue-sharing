## project description

通过Vault合同和由这些合同铸造的Vault NFT运作。
发送到Vault的资金在个人索赔之前平均分配给所有Vault NFT的所有者。
任何Vault NFT所有者都可以随时触发分配。第一个触发器为其他参与者支付Gas。
假设Vault收到了200美元USDC，
还假设您持有1000个NFT中的10个Vault NFT，
因此您将能够收到：200*(10/1000) = 2美元USDC
